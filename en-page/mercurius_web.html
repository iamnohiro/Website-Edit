Mercurius Web

Mercurius Web is a user-friendly and robust business management platform that enables customers to securely manage wireless devices — activations, usage and more. It is the industry's most comprehensive interface for ordering, activating and managing wireless services for M2M devices. Mercurius Web can be easily integrated into the business logic of customer's applications through an API that supports XML Web-services.

Mercurius Web also allows for threshold monitoring of single assets for unexpected, excessive usage or fraudulent use. Using this application, customers can obtain billing and usage information to optimize service delivery, thus lowering the total cost of ownership of your M2M ecosystem.

Mercurius Web Highlights:

  • One stop M2M management portal for all wireless network requirements.
  • Timestamp reporting of devices last known location and last session information.
  • Reporting, billing, and network management in one interface.
  • Easy access to business critical usage reporting, invoice and billing information developed
    specifically for the M2M market.
  • Reduce time, effort, and cost. Minimize provisioning errors of wireless device inventories.
