Mercurius Web

A Mercurius Web é uma plataforma de gestão de negócios de fácil utilização e robusta, que permite aos clientes gerenciar com segurança os dispositivos sem fio — ativações, utilização e muito mais. É a interface mais abrangente da indústria para encomendar, ativar e gerenciar serviços sem fio para dispositivos M2M. A Mercurius Web pode ser facilmente integrada na lógica de negócios de aplicativos do cliente através de uma API que suporta serviços web XML.

A Mercurius Web também permite o monitoramento de limite de ativos únicos para uso inesperado, excessivo ou fraudulento. Ao usar este aplicativo, os clientes podem obter faturamento e informações de uso para otimizar a prestação de serviços, reduzindo assim o custo total de ser proprietário de seu ecossistema M2M. 

Destaques da Mercurius Web:

  • Portal de gerenciamento M2M concentrado para todos os requisitos de rede sem fio.
  • Relatórios com carimbo de data/hora da última localização conhecida e da última sessão
    de informações dos dispositivos.
  • Relatórios, faturamento e gerenciamento de rede em uma única interface.
  • Fácil acesso a relatórios de utilização crítica do negócio, informações de fatura e
    faturamento desenvolvidas especificamente para o mercado M2M.
